Algorithm

  1. Patient enters clinic.
  2. Patient info and purpose for visit is taken by receptionist and checks the records, If patient has no records in clinic, create new record and save in database. Then give queue number to patient.
  3. Patient waits until provided queue number is called. If called proceed to doctor/attending physician for checkup.
  4. Patient receives summary of checkup, diagnosis and doctor's prescription.
  5. Patient pays for the fees and leaves.

Pseudocode

  • BEGIN patient arrives for clinical care
  • INPUT patient gives personal information and purpose for visit
    • IF patient given info has a match in database proceed to queue
    • ELSE submit personal information to database for record
    • THEN give queue number
    • ENDIF
  • OUTPUT queue number
    • IF queue number called proceed to checkup
    • ELSE wait for number to be called
    • REPEAT UNTIL number is called.
    • ENDIF
  • INPUT doctor/physician conducts checkup
  • OUPUT patient receives diagnosis and prescription.
  • INPUT patient pays for the fees
  • OUTPUT patient receives receipt for payment and leaves clinic
  • END
